HTML

  • Always starts with doctype html declaration
  • Head element
    • Contains information about the website
    • Not visible for the user
    • eg. title, meta, link elements
  • Body element
    • Contains the visible elements of the website
    • Divided into header, main and footer sections

CSS

  • Types: inline CSS, internal style sheet, external style sheet
  • External CSS file has to be linked to HTML with "link" element
  • CSS box model: margin > border > padding > content
  • margin: space around the element
  • padding: space acound the content inside the element
  • to use class selector use .class-name
  • box-shadow: x-offset y-offset blur spread-radius color

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Arrays are special variables that can hold more than value
  • Loops can execute a block of code a number of times
  • Functions are executed when they are called
